# belajar-python-server

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

Repository pembelajaran Python server-side programming.


## Instalasi

```bash
# Clone repository
git clone https://github.com/AryaFatthurahman1/belajar-python-server.git
cd belajar-python-server

# Buat virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt

# Jalankan aplikasi
python main.py
```
